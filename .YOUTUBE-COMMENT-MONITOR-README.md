## 📊 What It Does (Every 30 Minutes)

1. **Fetches** new comments from Concessa Obvius YouTube channel
2. **Categorizes** each comment:
   - 🤔 **Questions** (how, what, cost, timeline, tools) → Auto-respond
   - 👍 **Praise** (amazing, inspiring, awesome) → Auto-respond
   - 🚫 **Spam** (crypto, mlm, forex, "dm me") → Flag only
   - 🤝 **Sales** (partnership, collaboration) → Flag for review

3. **Auto-responds** to questions and praise with helpful templates
4. **Flags** spam and sales inquiries for manual review
5. **Logs everything** to `.cache/youtube-comments.jsonl` (JSONL format)
6. **Reports** statistics: total processed, auto-responses sent, flagged items

## 📊 Data Format (JSONL)

Each line is a JSON object:
```json
{
  "timestamp": "2026-04-16T20:00:50.647113",
  "comment_id": "demo_q1_1776394850645696",
  "video_id": "demoVideo1",
  "author": "Curious Cat",
  "text": "How do I start building my own system like this?",
  "category": "questions",
  "subcategory": "how_start",
  "auto_replied": true,
  "response_sent": "Great question! Start with ONE task..."
}
```

Parse it:
```bash
# Last 5 comments
tail -5 .cache/youtube-comments.jsonl | jq '.'

# All questions
jq 'select(.category=="questions")' .cache/youtube-comments.jsonl

# Comments requiring action (auto_replied=false)
jq 'select(.auto_replied==false)' .cache/youtube-comments.jsonl
```

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| **Cron not running** | Run `crontab -l` to verify it's installed. Check `launchctl list` on macOS. |
| **"No new comments"** | First run needs YouTube login. Run manually once to authenticate. |
| **YouTube API error** | Token may have expired. Delete `.secrets/youtube-token.json` and run again. |
| **Can't find script** | Make sure you're in `/Users/abundance/.openclaw/workspace` before running. |
| **Permission denied** | Run `chmod +x scripts/youtube-monitor-cron.sh` |

→ Make sure `auto_replied` should be `true` in JSONL. Check script for errors: `python3 scripts/youtube-comment-monitor.py`
