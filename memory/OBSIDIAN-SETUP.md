# Obsidian Setup Guide

## What You're Setting Up

A **knowledge graph** that shows how all your business systems interconnect:
- System identity
- Current status of all projects
- Decision history with rationale
- Daily learnings

Think of it as a visual map of the entire business + AI system.

---

## STEP 1: Download Obsidian (2 minutes)

1. Go to https://obsidian.md/
2. Download for Mac
3. Install (drag to Applications)
4. Open Obsidian

---

## STEP 2: Open the Vault (1 minute)

1. Click "Open Folder as Vault"
2. Navigate to: `/Users/abundance/.openclaw/workspace/memory/`
3. Select that folder
4. Obsidian opens the vault

---

## STEP 3: Explore the Graph

You're now in the Obsidian vault. Here's what you see:

### Left Sidebar
- **File explorer**: All `.md` files in the memory folder
- Click any file to open it

### Main Editor
- Markdown files (readable text with links)
- Click `[[filename]]` links to jump to other notes

### Right Sidebar
- **Graph View** (top-right icon, looks like network)
- Click to see visual knowledge graph
- Each node = a file
- Lines = connections between files
- Zoom/pan to explore

---

## Key Files to Know

- **INDEX.md** — Start here. Overview of everything.
- **abundance-context.md** — Who Abundance is, what the mission is
- **status-tracker.yaml** — Machine-readable current state (what's running, what's blocked)
- **session-log.md** — Every decision with rationale (why we did what we did)
- **2026-04-09.md** — Today's notes and learnings

---

## How the Graph Works

### What You See
- **Nodes** (circles) = Each markdown file
- **Lines** = Connections (when one file links to another)
- **Color** = Type of file or importance

### Click & Explore
- Click any node to jump to that file
- Zoom with scroll wheel
- Drag to pan around
- Bigger nodes = more connections (more important files)

---

## Tips

- **Don't edit YAML directly** — `status-tracker.yaml` is for machines to read. Use the plain `.md` files for human reading.
- **Add links freely** — Use `[[filename]]` to create connections. Graph updates automatically.
- **Dark/light theme** — Top-left corner menu, scroll to "Appearance"
- **Search** — Ctrl+P (or Cmd+P on Mac) to search all files
