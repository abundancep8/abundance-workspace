## 🎬 Overview

The YouTube DM Monitor is a production-ready system that:

1. **Monitors** YouTube DMs hourly from Concessa Obvius channel
2. **Categorizes** each DM into one of 4 categories
3. **Auto-responds** with templated messages
4. **Logs** all activity with timestamps
5. **Flags** interesting partnerships for manual review
6. **Reports** metrics on demand and scheduled

**Schedule:** Every hour (0 * * * *)  
**Log Location:** `.cache/youtube-dms.jsonl`  
**Report Location:** `.cache/youtube-dm-report.json`

## 🎯 Categorization System

### Category 1: Setup Help
**Keywords:** setup, how to, confused, beginner, tutorial, install, getting started, doesn't work, help, guide, stuck, error, not working

**Trigger:** User asks "how" or reports issues  
**Auto-Response:** Links to setup guide, FAQ, video  
**Action:** Auto-send + Log

### Category 2: Newsletter
**Keywords:** newsletter, updates, email list, subscribe, news, latest, stay updated, follow, sign up

**Trigger:** User wants to stay informed  
**Auto-Response:** Newsletter signup CTA with benefits  
**Action:** Auto-send + Log

### Category 3: Product Inquiry
**Keywords:** buy, pricing, price, cost, purchase, how much, afford, product, which version, recommend, features, difference, plan

**Trigger:** User wants to buy or learn features  
**Auto-Response:** Product info + follow-up questions  
**Action:** Auto-send + Log + Track for conversion

### Category 4: Partnership
**Keywords:** collaborate, sponsorship, partner, joint, co-brand, affiliate, promotion, promote, work together, business opportunity, brand deal

**Trigger:** User proposes business deal  
**Auto-Response:** Redirect to partnership email  
**Action:** Auto-send + Log + **FLAG for manual review**

**Flag Criteria:** Partnership is flagged if:
- Message > 150 characters (detailed inquiry)
- Contains: brand, major, large, budget, contract, deal
- Mentions YouTube, creator, influencer, agency
- Asks a specific question

## ⚙️ Installation Checklist

### Prerequisites
- [x] Python 3.7+
- [x] Playwright (install with: `python3 -m pip install --user playwright`)
- [x] Chromium browser (install with: `python3 -m playwright install chromium`)
- [x] YouTube account with Studio access
- [x] Bash/cron for scheduling

### Setup Steps
1. **Install Playwright:**
   ```bash
   python3 -m pip install --user playwright
   python3 -m playwright install chromium
   ```

2. **Verify installation:**
   ```bash
   python3 youtube-dm-monitor-live.py --test
   ```
   Expected: 4 test DMs categorized successfully

3. **Customize templates:**
   - Edit `youtube-dm-monitor-live.py`
   - Search for `TEMPLATES` dict
   - Update URLs and email addresses

4. **Test with real YouTube DMs:**
   ```bash
   python3 youtube-dm-monitor-live.py --report
   ```

5. **Activate cron scheduler:**
   ```bash
   crontab -e
   # Add: 0 * * * * /Users/abundance/.openclaw/workspace/cron-youtube-dm-monitor-live.sh
   ```

6. **(Optional) Set up Discord webhook:**
   ```bash
   export YOUTUBE_MONITOR_WEBHOOK="https://discord.com/api/webhooks/YOUR_ID"
   echo 'export YOUTUBE_MONITOR_WEBHOOK="..."' >> ~/.bash_profile
   ```

## 🚀 How to Use

### Manual Run
```bash
# Test mode (no browser)
python3 youtube-dm-monitor-live.py --test

# Run with report (opens browser, fetches real DMs)
python3 youtube-dm-monitor-live.py --report

# Headless mode (no window)
python3 youtube-dm-monitor-live.py --headless --report

# Debug mode
python3 youtube-dm-monitor-live.py --debug --report
```

### View Results
```bash
# Watch logs in real-time
tail -f ~/.cache/youtube-dm-monitor.log

# View recent DMs
tail -20 ~/.cache/youtube-dms.jsonl | jq '.'

# View partnerships flagged for manual review
grep '"interesting_partnership": true' ~/.cache/youtube-dms.jsonl | jq '{sender: .sender, text: .text}'

# Count by category
jq '.category' ~/.cache/youtube-dms.jsonl | sort | uniq -c

# Generate daily report
python3 youtube-dm-monitor-live.py --report --hours 24
```

### Scheduled Execution
```bash
# View crontab
crontab -l

# Edit crontab
crontab -e

# Stop monitoring
crontab -r  # (deletes all cron jobs!)
# Or just remove the YouTube line from: crontab -e
```

## 📊 Metrics & Reporting

### Daily Summary
```bash
python3 << 'EOF'
import json
from datetime import datetime, timedelta
from pathlib import Path

log_file = Path(".cache/youtube-dms.jsonl")
today = datetime.now().date()

dms = []
with open(log_file) as f:
    for line in f:
        dm = json.loads(line)
        dm_date = datetime.fromisoformat(dm['timestamp']).date()
        if dm_date == today:
            dms.append(dm)

print(f"📊 Daily Report for {today}")
print(f"Total DMs: {len(dms)}")
print(f"By category: {dict.fromkeys([d['category'] for d in dms])}")
print(f"Partnerships flagged: {sum(1 for d in dms if d.get('interesting_partnership'))}")
EOF
```

### Monthly Analytics
```bash
# Export to CSV
python3 << 'EOF'
import json, csv
from pathlib import Path

dms = []
with open(".cache/youtube-dms.jsonl") as f:
    for line in f:
        dms.append(json.loads(line))

with open("youtube-dms-export.csv", "w", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=['timestamp', 'sender', 'category', 'text'])
    writer.writeheader()
    writer.writerows(dms)

print(f"✅ Exported {len(dms)} DMs to youtube-dms-export.csv")
EOF
```

## 🆘 Troubleshooting

### Issue: Playwright not installed
**Solution:**
```bash
python3 -m pip install --user playwright
python3 -m playwright install chromium
```

### Issue: DMs not fetching
**Cause:** Browser session not authenticated or channel not accessible  
**Solution:**
```bash
# Run with debug to see errors
python3 youtube-dm-monitor-live.py --debug --report

# May need to manually log into YouTube first in browser
```

### Issue: Cron not running
**Debug:**
```bash
# Check crontab
crontab -l

# Make script executable
chmod +x cron-youtube-dm-monitor-live.sh

# Check system cron logs
log stream --predicate 'process == "cron"'

# Run script manually
/Users/abundance/.openclaw/workspace/cron-youtube-dm-monitor-live.sh
```

### Issue: Discord webhook not working
**Test:**
```bash
curl -X POST "$YOUTUBE_MONITOR_WEBHOOK" \
  -H 'Content-Type: application/json' \
  -d '{"content":"Test message"}'
```

### Issue: Timestamps comparison error
**Solution:** Delete `.cache/youtube-dms.jsonl` and restart
