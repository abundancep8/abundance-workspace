# 90-Day Campaign Implementation Plan

## Phase 1: Setup & Targeting (Weeks 1-2)

### Week 1 Actions

**LinkedIn Profile Optimization**
- [ ] Update LinkedIn headline to include practice-focused keyword
  - Example: "Helping Medical Practices Grow Revenue | Practice Growth Consultant"
- [ ] Update about section with specific practices you serve
- [ ] Add case studies/results to featured section
- [ ] Profile photo: professional, approachable
- [ ] Complete all profile sections

**Sales Navigator Setup** (if using)
- [ ] Purchase 3-month subscription ($99-499/month depending on plan)
- [ ] Set up 5 saved searches with your target criteria:
  1. Medium-sized medical practices (5-15 doctors)
  2. Dental practices
  3. Chiropractic practices
  4. Practice managers in Tier 1 metros
  5. Practice owners with recent promotions

**Prospect List Building**
- [ ] Manually build initial target list: 100 prospects (50 owners, 50 managers)
- [ ] Document:
  - Name, title, practice name
  - Practice size (# of doctors)
  - Location
  - Key pain point (based on LinkedIn profile)
  - Warm intro possibility (yes/no)
- [ ] Segment into Tier 1 (warm intro), Tier 2 (personalized), Tier 3 (template)

---

### Week 2 Actions

**Message Template Library**
- [ ] Write 3 versions of connection request message
  - Version A: Highly personalized (for Tier 1)
  - Version B: Semi-personalized (for Tier 2)
  - Version C: Template (for Tier 3)
- [ ] Write 2 follow-up sequences (for non-respondents)
- [ ] Write 2 objection response templates

**Content Calendar Planning**
- [ ] Plan 8 LinkedIn articles for next 2 months (1-2 per week)
  - Topics drawn from pain points research
  - Examples: Revenue optimization, staffing, patient experience, etc.
- [ ] Outline first 2 articles
- [ ] Schedule 3 LinkedIn posts (industry news, practice tips, stat-sharing)

**Team Training** (if applicable)
- [ ] Brief team on messaging strategy
- [ ] Align on talking points for objections
- [ ] Establish tracking/CRM system for outreach

---

## Phase 2: Warm Outreach & Authority Building (Weeks 3-5)

### Week 3 Actions

**Warm Introduction Campaign**
- [ ] Identify 20-30 prospects where you have warm intro possibility
- [ ] Reach out to mutual connections requesting intro
  - Message template: "I'm trying to connect with [Name] at [Practice]. Do you know them? Would you mind intro-ing us?"
- [ ] Expected result: 5-10 warm intros generated
- [ ] Send messages to warm-intro targets with context

**Content Publishing (Article 1)**
- [ ] Publish first LinkedIn article
  - Topic: Top pain point for your target audience
  - Length: 800-1200 words
  - Include 1-2 specific metrics/case study
  - CTA: "Comment if this resonates"

**Profile Engagement Strategy**
- [ ] Engage with 10 target prospects' recent posts
  - Like, comment thoughtfully (not "great post!")
  - This builds familiarity before outreach

---

### Week 4 Actions

**Cold Outreach Campaign - Tier 2 (Personalized)**
- [ ] Send connection requests to 40 Tier 2 prospects
  - Use semi-personalized message template
  - Customize 2-3 details (practice name, location, specific pain point)
- [ ] Track: sent date, response status
- [ ] Expected result: 30% response rate (12 connections)

**Follow-up on Warm Intros**
- [ ] Send first message to 5-10 warm intro targets
  - Thank mutual connection in message
  - Reference specific context from their practice
  - Ask for brief call (15-20 min)
- [ ] Expected result: 35-40% response rate (2-4 calls scheduled)

**Content Publishing (Article 2)**
- [ ] Publish second article
  - Different pain point than Week 3
  - Include competitive benchmark data

**Podcast/Interview Outreach** (Optional)
- [ ] Identify 10 healthcare podcasts targeting practice owners
- [ ] Pitch yourself as guest expert (1-2 per month)

---

### Week 5 Actions

**Tier 1 Personalized Campaign**
- [ ] Send handwritten/hyper-personalized connection requests to 20 Tier 1 prospects
  - 5+ minutes research per prospect
  - Reference specific practice news (expansion, award, etc.)
  - Position as peer/advisor, not salesperson
- [ ] Send 1st follow-up message to Week 4 connections (if no response)
- [ ] Expected result: 40-50% response rate (8-10 connections)

**Case Study Development**
- [ ] Create 3 detailed case studies (if you have clients)
  - Format: Challenge → Solution → Results
  - Include specific metrics
  - Get client approval for LinkedIn sharing
- [ ] Turn into downloadable PDFs for sharing with prospects

**Weekly Assessment**
- [ ] Track metrics:
  - Connections sent: 60
  - Connections accepted: 18
  - Messages sent: 12
  - Calls scheduled: 4
  - Articles published: 2
  - Engagement (likes + comments): 50+

---

## Phase 3: Scale & Optimization (Weeks 6-8)

### Week 6 Actions

**Tier 3 Template Campaign Launch**
- [ ] Send connection requests to 100 Tier 3 prospects
  - Use template messages (minimal personalization)
  - Target: 15-20% connection rate (15-20 connections)
- [ ] Send first message follow-ups to Tier 2/3 connections
  - Reference their industry/pain point
  - Offer value (article, case study, consultation)
  - Soft CTA: "Worth a 15-min call?"

**LinkedIn Ads Test** (Optional but Recommended)
- [ ] Create 3 ad campaigns targeting different segments:
  - Segment A: Practice owners, medium groups, $200K+ budget
  - Segment B: Practice managers, all sizes
  - Segment C: Warm audience (website visitors, email list)
- [ ] Budget: $500-1000 for test
- [ ] Goal: Drive to LinkedIn article or landing page
- [ ] Expected result: 50-100 website clicks, 5-10 conversations

**Content Publishing (Articles 3-4)**
- [ ] Publish 2 more articles
  - Topics: Different pain points or case studies
  - Cross-promote with connections

**Call Preparation**
- [ ] Develop 15-min discovery call script
  - Pain point validation questions
  - Fit assessment questions
  - Next-step options (more detailed call, proposal, etc.)
- [ ] Train sales team on call script

---

### Week 7 Actions

**Follow-up Sequences Execution**
- [ ] Send 2nd message to Week 4-5 non-respondents
  - Different angle or pain point
  - Reference specific content/insight
- [ ] Send 1st message to Week 6 connections
- [ ] Expected result: 5-10 additional calls scheduled from follow-ups

**Competitor Intelligence Gathering**
- [ ] Monitor what 3 main competitors are posting about
- [ ] Note messaging themes and engagement
- [ ] Update competitive positioning statement

**Community Building**
- [ ] Start commenting on 20 target prospects' posts weekly
  - Thoughtful, relevant comments (not promotional)
  - Build relationship over time
- [ ] Share 5-10 other experts' content to position as curator

**LinkedIn Sales Navigator Insight Review**
- [ ] Analyze: Which searches gave best response rate?
- [ ] Which roles (owner vs. manager) respond better?
- [ ] Which geographies respond better?
- [ ] Adjust targeting based on data

---

### Week 8 Actions

**Campaign Momentum Building**
- [ ] Review all metrics to date
- [ ] Identify top 10 warm prospects for phone outreach
- [ ] Schedule calls directly (don't ask permission)
- [ ] Expected activity:
  - Outreach total: 200+ prospects contacted
  - Conversations scheduled: 10-15
  - Article performance: 200+ views, 30+ comments
  - LinkedIn network growth: +150-200 connections

**Referral Program Launch**
- [ ] Reach out to 5-10 satisfied customers
- [ ] Ask for referral introductions to similar practices
- [ ] Offer referral incentive (discount, service credit, etc.)

**Website/Landing Page Update**
- [ ] Create dedicated landing page for medical practice outreach
  - Highlight case studies/results
  - Capture email for follow-up
  - Optimize for practice owner keywords
- [ ] Add LinkedIn follow button to website

---

## Phase 4: Optimization & Growth (Weeks 9-12)

### Weeks 9-10 Actions

**Qualitative Feedback Loop**
- [ ] Review all calls/conversations to date
- [ ] Identify 3 most common objections
- [ ] Create response framework for each
- [ ] Update messaging based on what gets best response

**Persona Refinement**
- [ ] Update ideal customer profile based on best respondents
  - Practice size, location, pain point
  - Who are your easiest conversions?
- [ ] Shift 70% of effort to targeting similar profiles

**Content Performance Analysis**
- [ ] Review which articles got best engagement
- [ ] Which pain points resonated most?
- [ ] Plan next 4 articles based on top performers

**LinkedIn Ads Scaling**
- [ ] If Ads test performed (5%+ CTR, <$20 CPC):
  - Increase budget 2-3x
  - Test new audience combinations
  - Run carousel ads with case studies
- [ ] If underperformed:
  - Test new creative/messaging
  - OR reallocate to organic outreach

---

### Weeks 11-12 Actions

**Sales Pipeline Maturation**
- [ ] Expected outcome: 5-10 qualified opportunities in early stage
- [ ] Map out sales cycle (typically 30-90 days for practices)
- [ ] Create proposal templates for specific use cases

**Referral Program Acceleration**
- [ ] If generating referrals, expand to 20+ referral partners
- [ ] Create co-marketing opportunities

**Next Quarter Planning**
- [ ] Analyze 90-day results:
  - Cost per conversation
  - Conversation to close rate
  - Customer acquisition cost (CAC)
  - LTV (if you have contracts)
- [ ] Plan Q2 strategy based on learnings

## Key Metrics to Track

### Engagement Metrics
- Connections sent/accepted
- Profile views
- Article views
- Article comments
- LinkedIn post engagement

### Conversion Metrics
- Messages sent → calls scheduled (conversion %)
- Calls scheduled → meetings held (show-up rate)
- Meetings → proposals (interest rate)
- Proposals → closed deals (close rate)

### Cost Metrics
- LinkedIn ads spend
- Cost per click
- Cost per conversation
- Cost per qualified lead
- Cost per customer acquisition

### Content Metrics
- Article views
- Article comments/shares
- LinkedIn post impressions
- LinkedIn post engagement rate

## Risk Mitigation

| Risk | Mitigation |
|------|-----------|
| Low response rates | Improve personalization; test different pain points |
| Wrong audience targeting | Gather feedback, refine buyer persona |
| Messaging not resonating | A/B test messaging; update objection responses |
| Implementation burden | Automate with CRM/LinkedIn scheduling tools |
| Competitor stealing prospects | Move fast; establish thought leadership |
| Decision cycle longer than expected | Involve practice manager (faster decision) |

## Tools & Resources

**Recommended tools:**
- LinkedIn Sales Navigator ($500-1K/month) - Essential for scale
- CRM (Pipedrive, HubSpot, Salesforce) - Track conversations
- Email tracking (Superhuman, Mailtrack) - See who reads your emails
- LinkedIn post scheduler (Buffer, Hootsuite) - Consistent publishing
- Calendly - Make scheduling friction-free

**Budget estimate:**
- LinkedIn ads: $500-2,000/month (optional, test first)
- Tools: $300-800/month
- Content creation (if outsourced): $500-2,000/month
- **Total: $1,300-4,800/month**

---

## Success Criteria (90 Days)

**Baseline success:**
- 300+ contacts outreached
- 20-30 qualified conversations
- 2-4 closed deals
- Established thought leadership (50+ article views/week)

**Stretch success:**
- 500+ contacts
- 40-50 conversations
- 8-10 closed deals
- 500+ LinkedIn followers

**Red flags (pivot if occurring):**
- <5% response rate after 100 contacts
- <20% of conversations convert to calls
- <10% of calls result in interest
- Articles getting <50 views
