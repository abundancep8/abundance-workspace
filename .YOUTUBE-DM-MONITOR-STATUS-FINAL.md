## 📞 Support

**Issue: "No DMs found"**
→ First run requires YouTube login in the browser. Run manually once to authenticate.

**Issue: "Can't find YouTube Studio messages"**
→ Check that DMs actually exist in your YouTube Studio. Monitor can only see DMs that are there.

**Issue: "Cron not running"**
→ Check: `crontab -l | grep youtube` or `launchctl list | grep youtube`

**Issue: "Virtual environment not found"**
→ Run this to regenerate: `python3 -m venv ~/.openclaw/workspace/venv`
