## 🎯 Categorization Logic

The monitor automatically categorizes each comment into one of four categories:

| Category | Keywords | Action |
|----------|----------|--------|
| **Questions** | how, what, why, when, cost, price, timeline, tools, start | ✅ Auto-respond with helpful answer |
| **Praise** | amazing, awesome, inspiring, love, great, thank you, brilliant | ✅ Auto-respond with gratitude |
| **Spam** | crypto, bitcoin, nft, mlm, forex, "dm me" | 🚫 Log & flag as spam |
| **Sales** | partnership, collaboration, sponsorship, affiliate, business opportunity | 🚩 Flag for manual review |

## 🔧 Cron Setup

### Option 1: Add to Crontab (Recommended)

```bash
crontab -e
# Add this line:
*/30 * * * * /Users/abundance/.openclaw/workspace/scripts/youtube-monitor-cron.sh >> /Users/abundance/.openclaw/workspace/.cache/youtube-cron-exec.log 2>&1
```

### Option 2: Use LaunchD (macOS Alternative)

Create `~/Library/LaunchAgents/com.concessa.youtube-comment-monitor.plist`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.concessa.youtube-comment-monitor</string>
    <key>ProgramArguments</key>
    <array>
        <string>/Users/abundance/.openclaw/workspace/scripts/youtube-monitor-cron.sh</string>
    </array>
    <key>StartInterval</key>
    <integer>1800</integer>
    <key>StandardOutPath</key>
    <string>/Users/abundance/.openclaw/workspace/.cache/youtube-cron-exec.log</string>
    <key>StandardErrorPath</key>
    <string>/Users/abundance/.openclaw/workspace/.cache/youtube-cron-error.log</string>
</dict>
</plist>
```

Then load it:
```bash
launchctl load ~/Library/LaunchAgents/com.concessa.youtube-comment-monitor.plist
```

## 📞 Troubleshooting

| Issue | Solution |
|-------|----------|
| **"No new comments found"** | First run needs YouTube login. Run manually once. |
| **"YouTube API libraries not available"** | Install: `pip install google-api-python-client google-auth-oauthlib` |
| **"Token invalid"** | Delete `.secrets/youtube-token.json` and run script again to re-authenticate. |
| **"Cron not running"** | Check: `crontab -l` or `launchctl list \| grep youtube` |
| **"Too many requests (429)"** | YouTube API rate limited. Monitor will retry automatically. |
