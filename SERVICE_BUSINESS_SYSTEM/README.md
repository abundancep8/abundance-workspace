## Automation Flows

**Lead Gen Flow (Daily):**
1. Scrape 50 qualified LinkedIn profiles (medical/B2B owners)
2. Auto-score based on fit criteria
3. Add to outreach queue
4. Send templated connection requests

**Sales Flow (Automated):**
1. Qualified leads get auto-email sequence
2. Auto-schedule demo at Calendly link
3. Send demo video 24h before call
4. Auto-send proposal after call
5. Track pipeline status in dashboard

**Delivery Flow (24h after deal closed):**
1. Client onboarding initiated
2. Intake form auto-sent
3. Schedule setup call
4. Deploy OpenClaw instance
5. Configure integrations
6. Go-live in 14 days

**Success Flow (Daily):**
1. Monitor client ROI metrics
2. Generate daily performance report
3. Auto-alert if issues detected
4. Trigger upsell when milestone hit

**Support Flow (24/7):**
1. Auto-respond to support tickets
2. Route to KB first
3. Escalate if needed
4. Monthly success review auto-scheduled
