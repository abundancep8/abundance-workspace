✓ Monitors your Concessa Obvius channel for new DMs
✓ Automatically categorizes each message:
  • 🆘 Setup Help (how-to, errors)
  • 📬 Newsletter (email subscriptions)
  • 🛍️ Product Inquiry (pricing, purchase)
  • 🤝 Partnership (sponsorships, collabs)

✓ Generates hourly reports with metrics:
  - Total DMs processed
  - Auto-responses sent
  - Breakdown by category
  - Partnerships flagged
  - Conversion potential

✓ Prevents duplicate processing with state tracking
✓ Runs automatically every hour via cron/launchd

Smart Categorization
  Uses keyword matching to automatically categorize each DM
  Customizable keywords in the config
  Easy to adjust category heuristics

Auto-Responses
  Sends templated replies for each category
  Response templates are customizable (edit youtube-dm-templates.json)
  Tracks which DMs were responded to

Partnership Flagging
  Marks partnerships with detailed pitches (100+ chars) 
  Flags those mentioning budget, brand, investment
  Flags "co-brand", "sponsorship", "revenue share" opportunities

Complete Logging
  Every DM is logged to .cache/youtube-dms.jsonl
  JSONL format (one JSON object per line)
  Append-only = never lose data
  Includes: timestamp, sender, text, category, response, flags

Hourly Reporting
  Generates metrics every hour
  Shows: total DMs, responses sent, category breakdown
  Reports conversion potential (product inquiry = possible sale)
  Emails flagged partnerships

State Management
  Tracks processed DM IDs to prevent duplicates
  Even if DM appears twice, processed only once
  Maintains run count and statistics
